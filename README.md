# AquaClean JAX - Hull Cleaning Services Website

## 🚀 Ready for Netlify Deployment!

Professional website for hull cleaning services in Jacksonville, FL and Northeast Florida.

### Quick Deploy to Netlify:

1. **Drag & Drop**: Drag the `netlify-deploy` folder to [netlify.com](https://netlify.com)
2. **Or use CLI**: `netlify deploy --prod --dir=netlify-deploy`
3. **Or connect Git**: Push to GitHub and connect via Netlify dashboard

### Features:
- ✅ Professional marine industry design
- ✅ Mobile responsive
- ✅ Contact form with Netlify Forms integration
- ✅ SEO optimized for local search
- ✅ Interactive FAQ accordion
- ✅ Performance optimized
